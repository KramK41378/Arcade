Название игры: Buckshot Blitz

Идея игры:
1) начальное окно, это просто заставка, там будет название игры, авторы и надпись нажмите лубую кнопку чтобы продолжить
2) дальше открывается "меню" с кнопками "режим уровня" и "выбрать персонажа"
3) по надатию кнопки выбрать персонажа откроется меню с моделями героев, кликаешь по герою и он у тебя выбирается
4) по нажатию кнопки режим уровня открывается окно с тремя кнопками(сложностью режима), с увеличением сложности увеличивается скорость противника
5) сами уровни. Это некие круглые арены где есть наш персонаж, он стреляет раз в некоторое время, например, раз в 0.5 секунд и по кругу спавнятся разные мобы(слизни, зомби, тп), скорость мобов зависит от сложности уровня. У персонажа есть хп, соответственно мобы наносят дамаг, при 0 хп игра заканчивается и финальное окно с результатом времени, окно можно скипнуть нажатием на лубую кнопку и мы вернемся в выбор уровня
6) Так же во время игры можно добавить счетчик времени, то есть наш результат и в окне, где мы выбираем уровень(пункт 4) рядом с кнопками выбора написать "Ваш рекорд: #" + можно хранить данные о рекорде в тхт или бд(это будет оцениваться)

Дополнительно мы добавим след классный персонажу, взрыв от пуль персонажей, 3-4 персонажа в меню выбора персонажей, 2-3 монстра, всем добавим классную анимацию и можно в меню что-то с задним фоном сделать, например, какую-нибудь картинку по игре меняющуюся. Можно еще добавить кнопки чтобы двигаться между менюшек
